# Styled Document Layout

This project is a simple web page built using HTML and CSS, focused on structuring and styling textual content in a clean and readable format.

## Purpose

The main goal of this project is to practice and demonstrate fundamental web development concepts, especially:

* HTML structure and semantic organization
* CSS styling and layout
* Typography and readability
* Separation of content and design

Instead of being just a basic exercise, this project represents an early step in building well-structured and visually organized web pages.

## Technologies Used

* HTML5
* CSS3

## Features

* Chapter-based content organization
* Styled headings (H1 and H2)
* Text alignment for better readability
* Custom color scheme
* External CSS for styling separation

## What I Learned

Through this project, I was able to:

* Apply CSS styles to improve visual hierarchy
* Understand how to organize content using headings
* Work with fonts, colors, and spacing
* Structure a simple multi-section page

## Future Improvements

This project can be expanded with:

* Responsive design (mobile-friendly layout)
* Improved color palette and design system
* Navigation between pages
* More advanced layout techniques (Flexbox / Grid)
